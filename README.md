# Merey Portfolio

This repository contains a personal portfolio website built with GitHub Pages and Markdown.

## Edit content

Each main section of the website lives in its own Markdown file:

- `index.md`
- `background.md`
- `education.md`
- `experience.md`
- `achievements.md`
- `demo-lessons.md`
- `projects.md`
- `references.md`

Update the text in those files and push the changes to republish the site.
